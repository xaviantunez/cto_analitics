Sistema de Estadísticas de Fútbol

Instrucciones:
1. Descomprima el archivo ZIP en una carpeta
2. Abra index.html en un navegador web moderno
3. Use las siguientes credenciales para el primer acceso:
   Usuario: admin
   Contraseña: admin123
4. Desde el panel de administración puede crear más usuarios y configurar equipos

Requisitos:
- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- JavaScript habilitado
- Resolución mínima recomendada: 320px (compatible con móviles)

Notas:
- Todos los datos se almacenan localmente en el navegador
- Para borrar todos los datos, limpie el almacenamiento local del navegador
